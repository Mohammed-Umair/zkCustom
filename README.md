# KeyCraft Customization Store

A React + TypeScript + Tailwind landing experience for custom keycaps where users can:

- Browse featured keycap sets on a landing page (hero slider, testimonials, reviews, gallery).
- Open a single product details page.
- Add custom preferences (theme, legends, icon style, quantity) to an order.
- Visit the order booking page to scan a QR code for payment and send booking details by email to the shop owner.

## Tech stack

- React
- TypeScript
- Tailwind CSS
- Vite

## Run locally

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Deploy on Vercel

1. Push this repo to GitHub.
2. Import repository in Vercel.
3. Use default framework preset: **Vite**.
4. Build command: `npm run build`
5. Output directory: `dist`
6. Deploy.

`vercel.json` already includes rewrite support for SPA routes.
